## 📌 Project Overview

This project analyses unemployment rate data from **May 2019 to June 2020** to:

- Quantify the **impact of COVID-19** on unemployment.
- Detect **seasonal patterns** (monthly/quarterly trends).
- Compare **rural vs. urban** unemployment.
- Identify **region‑wise disparities** during the pandemic.
- Build **predictive models** to forecast unemployment based on simple features.
- Provide **policy‑relevant insights** for economic recovery.

The dataset contains monthly unemployment figures for 28 Indian states/UTs, including rural/urban breakdowns.
